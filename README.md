# shoko-front

in Japanese

Shoko（書庫）は書籍の管理ツールです。

自分が持っている書籍を管理することはもちろんですが、所有最新巻・既刊最新巻を記録します。
こうすることで書店等で所有最新巻を見られるのでダブリ購入をすることが防げます。
